# Movilidad & Flexibilidad

App de ejercicios de movilidad y flexibilidad con:

- 🧭 Cuestionario personalizado (nivel, molestias, condiciones, objetivo)
- 🎯 Motor de recomendación inteligente
- 📋 23 ejercicios en 7 zonas del cuerpo
- ▶️ Vídeos guía de YouTube por ejercicio
- ⚙️ Variantes adaptadas por condición
- ⏱️ Temporizador integrado
- 📱 Modo guiado paso a paso

## Stack

- React + Vite
- CSS-in-JS inline styles

## Desarrollo

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
```
